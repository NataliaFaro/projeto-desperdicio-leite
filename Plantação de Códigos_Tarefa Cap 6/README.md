# Controle de Desperdício de Leite

## Problema
O projeto trata do desperdício de leite na produção rural. Muitas propriedades não possuem um controle organizado da produção diária e das perdas, dificultando a análise e a tomada de decisão.

## Solução
Foi desenvolvido um sistema em Python que permite cadastrar registros de produção de leite, calcular o percentual de desperdício e classificar o nível de perda (baixo, médio ou alto).

## Funcionalidades
- Cadastro de registros
- Listagem de registros
- Cálculo automático de desperdício
- Classificação do nível de perda
- Salvamento em arquivo texto
- Salvamento em JSON
- Gravação em banco Oracle

## Tecnologias utilizadas
- Python
- JSON
- Arquivo texto
- Oracle Database

## Estrutura do projeto
- Cap_6_Tarefa.py (código principal)
- registros.txt
- registros.json
- banco.sql

## Conclusão
O sistema auxilia o produtor rural a controlar e analisar o desperdício de leite, contribuindo para melhor gestão e tomada de decisão.
