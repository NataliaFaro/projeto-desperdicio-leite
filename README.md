## 📜 Descrição

Este projeto tem como objetivo auxiliar no controle do desperdício de leite na produção rural. Muitas propriedades não possuem um controle estruturado da produção diária e das perdas, o que dificulta a análise dos dados e a tomada de decisão.

Como solução, foi desenvolvido um sistema em Python que permite registrar a produção de leite, calcular automaticamente o percentual de desperdício e classificar o nível de perda em categorias como baixo, médio ou alto.

Além disso, o sistema possibilita o armazenamento das informações em diferentes formatos, como arquivos texto, JSON e banco de dados Oracle, garantindo maior flexibilidade e organização dos dados.

O projeto contribui para uma melhor gestão da produção, permitindo ao produtor identificar falhas, reduzir desperdícios e otimizar seus resultados.

## 🔧 Como executar o código

### Pré-requisitos:

* Python 3 instalado
* Biblioteca `oracledb` instalada
* Banco de dados Oracle configurado

### Passo a passo:

1. Baixe ou clone o repositório
2. Abra o projeto em uma IDE (ex: PyCharm ou VS Code)
3. Configure a conexão com o banco Oracle no código
4. Execute o arquivo principal:

   ```bash
   python Cap_6_Tarefa.py
   ```
5. Utilize o menu interativo para cadastrar, listar e analisar os registros
